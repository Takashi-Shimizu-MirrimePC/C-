*******１．４記号定数*******
***温度換算で使用していた300や20などの変わることのない数値を定数として用いる
***#define 名前　それと置き換えるテキスト
***記号名・記号定数として扱うことができる。
    コンパイラが指定した名前が出てくると対応する文字列に置き換える
    名前としては変数名と同じ型、すなわち英字の後に英数字を続けたものを書く
***#defineを用いたサンプルプログラム
    #include <stdio.h>
    #define     LOWER 0     //下限
    #define     UPPER 300   //上限
    #define     STEP 20     //刻み
    int main ()
    {
        int fahr;
        for ( fahr = LOWER; fahr <= UPPER; fahr = fahr + STEP)
            printf("%3d %6.1f\n", fahr, (5.0 / 9.0 ) * ( fahr - 32.0 ));
    }
***ここで、LOWER,UPPER,STEPという量は定数であって変数ではないから宣言しなくてよい
    記号名は小文字の変数名と区別できるように大文字で書くのが普通で㌃
    #defineのあとにはセミコロンは必要ないことに注意
